Weather Application

A simple weather application built with HTML, CSS, and vanilla JavaScript. 
The app fetches real-time weather data from a public API and displays current conditions based on user input.

Features
- Search weather by city
- Display temperature and weather conditions
- Dynamic updates without page reload

Technologies Used
- HTML
- CSS
- JavaScript
- Public Weather API

What I Learned
- Fetching data from external APIs
- Working with asynchronous JavaScript
- Updating the DOM dynamically
- Basic error handling
